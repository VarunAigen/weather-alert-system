## üîß **How to Download from LottieFiles**

1. Go to https://lottiefiles.com
2. Search for the animation (e.g., "sun weather")
3. Click on an animation you like
4. Click **"Download"** button
5. Select **"Lottie JSON"** format
6. Save to `mobile/assets/animations/` folder
7. Rename to match the names above
